Jupiter's Moon Io Going Into Eclipse


(Python/IDL)

Python and IDL pipelines were created in order to intake, wrangle, clean, analyze data.
This project utilizes astronomical data from 3 telescopic sites in order to determine the dependence of Io's atmosphere on interaction with the sun and the Io plasma torus, a region of energetic particles around Jupiter.
Presented at: EPSC 2020, APO Symposium
Paper: Coming Soon (2022)




EXPRES Calcium Pipeline for False Positive Detection


(Python)

A Python pipeline were created in order to wrangle, select, and analyze data, as well as to perform linear regression.
This project utilizes EXPRES instrument data to find calcium spectral line ratios in order to dectect magnetic activity in stars, which could be misinterpreted as false exoplanet detection. Incorporated into main pipeline of instrument.
Paper: arxiv
